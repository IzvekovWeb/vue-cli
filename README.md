# Vue-cli

## Установка проекта
```
npm install
```

### Компиляция и жесткая перезагрузка виртуального сервера для разработки
```
npm run serve
```

### Компиляция и минимизация на прод.
```
npm run build
```

***

## Что входит в сборку

- SASS
- PUG
- Vue Router
- Vuex
